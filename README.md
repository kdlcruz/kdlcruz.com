## Local Development Setup

* [Ionic Framework 4 and NPM](https://ionicframework.com/docs/)

## Get Started

1. Pull repo. `git clone git@github.com:kdlcruz/kdlcruz.com.git`
2. Go to project. `cd kdlcruz.com`
3. Install modules. `npm install`
4. Start project. `npm start`

## CI/CD

This project is heavily integrated with a CI/CD pipeline currently running on CircleCI. The app is deployed in Google App Engine.

## Deployment to Production

all merges to `master` will automatically deploys to production
